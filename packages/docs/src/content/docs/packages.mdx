## API Clients

The API clients expose Effect services around generated OpenAPI clients. They keep generated schemas available through subpath exports and provide `Context.Service` layers for application code.

| Package                        | Public exports                                                                          | Auth model                                                           |
| ------------------------------ | --------------------------------------------------------------------------------------- | -------------------------------------------------------------------- |
| `@knpkv/jira-api-client`       | `JiraApiClient`, `JiraApiConfig`, `FetchClientError`, generated `v3` types.             | Basic auth or OAuth2 through Atlassian cloud gateway with `cloudId`. |
| `@knpkv/confluence-api-client` | `ConfluenceApiClient`, `ConfluenceApiConfig`, generated `v1` and `v2` types.            | Basic auth or OAuth2 through Atlassian cloud gateway with `cloudId`. |
| `@knpkv/clockify-api-client`   | `ClockifyApiClient`, `ClockifyApiConfig`, `ClockifyApiError`, generated Clockify types. | `X-Api-Key` header.                                                  |

Minimal service shape:

```typescript
import { Effect, Layer } from "effect"
import * as Redacted from "effect/Redacted"
import { JiraApiClient, JiraApiConfig, toEffect } from "@knpkv/jira-api-client"

const program = Effect.gen(function* () {
  const jira = yield* JiraApiClient
  return yield* toEffect(jira.v3.client.GET("/rest/api/3/myself"))
})

const ConfigLive = Layer.succeed(JiraApiConfig, {
  baseUrl: "https://example.atlassian.net",
  auth: {
    type: "basic",
    email: "you@example.com",
    apiToken: Redacted.make("...")
  }
})

Effect.runPromise(program.pipe(Effect.provide(JiraApiClient.layer), Effect.provide(ConfigLive)))
```

## Shared Libraries

| Package                   | Purpose                                                                                                                 |
| ------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| `@knpkv/atlassian-common` | Branded ids, hashing, OAuth operations, XDG config/token storage, AST nodes, and Markdown serialization.                |
| `@knpkv/codecommit-core`  | CodeCommit domain models, AWS client, config service, cache service, sandbox service, stats, and pull request services. |
