---
title: Migration
description: Moving from legacy command spellings to canonical commands.
---

Legacy command spellings are removed rather than kept as compatibility aliases.

| Legacy                     | Canonical                    |
| -------------------------- | ---------------------------- |
| `jira get`                 | `jira issue get`             |
| `jira search`              | `jira issue search`          |
| `jira version view`        | `jira version get`           |
| `jira version set`         | `jira version update`        |
| `jira version relatedwork` | `jira version related-work`  |
| `confluence clone`         | `confluence workspace clone` |
| `confluence pull`          | `confluence sync pull`       |
| `confluence push`          | `confluence sync push`       |
| `confluence fetch`         | `confluence page get`        |
| `jcf start`                | `jcf timer start`            |
| `jcf list`                 | `jcf issue list`             |
| `jcf reconcile`            | `jcf sync reconcile`         |
